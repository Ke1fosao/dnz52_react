# Сайт ЗДО №52 м. Рівне

Веб-сайт закладу дошкільної освіти №52, розроблений на Django у рамках дипломної роботи.

🌐 **Live:** https://ke1fosao.pythonanywhere.com

---

## 📋 Опис системи

Інформаційно-довідковий сайт дитячого садка з повноцінною адмінпанеллю
Django, переробленою під людей без технічного досвіду (кольорові секції з
емодзі, дружня стартова сторінка з підказками). Призначений для батьків,
педагогів та адміністрації закладу.

### Основні розділи

- **🏠 Головна** — слайдер (адаптивний на мобільному), останні новини, переваги закладу, швидкий пошук
- **📰 Новини** — публікації з категоріями, пагінацією та лічильником переглядів
- **📄 Про заклад / Педагогам / Батькам** — статичні сторінки з WYSIWYG-редактором (CKEditor)
- **🏘️ Групи** — інформація про всі вікові групи (1–14) з вихователями та помічниками; випадаюче меню «Групи» у навбарі з вертикальним прокручуванням на мобільному
- **👩‍🏫 Спеціалісти** — методист, психолог, музкерівник, медсестра, фізкультурний керівник; фото з адаптивним відображенням на мобільному; підписи фото обрізаються до 2 рядків
- **🎨 Гурткова робота** — описи гуртків з керівниками, розкладом, кольоровими номерами, статистикою закладу та CTA-банером
- **🍽️ Меню харчування** — щоденне меню (сніданок / II сніданок / обід / полуденок / вечеря); кольорова картка «Сьогодні» + тиждень Пн-Пт з навігацією попередніх/наступних тижнів; в адмінці одна форма на день + дія «Скопіювати на наступний тиждень»
- **📜 Атестація педагогів** — документи атестаційної комісії, етапи, кваліфікаційні категорії, нормативна база — все редагується через адмінку
- **🖼️ Фотогалерея** — альбоми згруповані за категоріями (Групи / Свята / Заняття / Літо / Спорт / Інше) з чіпами-фільтрами; lightbox для перегляду; на сторінці «Батькам» фото також відкриваються у lightbox
- **📁 Документи** — нормативні документи з категоріями що згортаються/розгортаються; дві кнопки «Переглянути» (у браузері) і «Завантажити»
- **💬 Відгуки** — з модерацією, рейтингом, статистикою, пагінацією та подвійним захистом від спаму (honeypot + rate-limit)
- **📞 Контакти** — адреса, телефони (кожен як окреме клікабельне посилання), email, режим роботи; iframe Google Maps на всю ширину; швидкі дії «Зателефонувати / Написати / Прокласти маршрут»
- **🎓 Керівництво** — окрема сторінка з директором та персоналом
- **🔍 Пошук** — глобальний пошук по новинах, сторінках, документах, спеціалістах, групах, гуртках
- **🗺 Sitemap.xml + robots.txt** — для SEO; підключено через `django.contrib.sitemaps`

- **python-dotenv** — змінні оточення

## 🚀 Встановлення (Windows)

### 1. Розпакувати проєкт і відкрити PowerShell у папці `dnz52/`

### 2. Створити віртуальне середовище

```powershell
python -m venv .venv
.venv\Scripts\activate
```

### 3. Встановити залежності

```powershell
pip install -r requirements.txt
```

### 4. Створити файл `.env` з конфігурацією

```powershell
copy .env.example .env
```

Згенерувати секретний ключ:

```powershell
python -c "from django.core.management.utils import get_random_secret_key; print(get_random_secret_key())"
```

Відкрити `.env` у редакторі і вставити отриманий ключ у `SECRET_KEY=...`.

### 5. Застосувати міграції БД

```powershell
python manage.py migrate
```

Створяться всі таблиці + автоматично заповняться початкові дані (категорії
галереї, документи атестаційної комісії тощо).

### 6. Створити суперкористувача (для адмінки)

```powershell
python manage.py createsuperuser
```

### 7. Запустити сервер

```powershell
python manage.py runserver
```

Сайт буде доступний на http://127.0.0.1:8000/
Адмінка — http://127.0.0.1:8000/admin/

## 🎓 Дружня адмінка

Стандартна Django-адмінка перероблена під людей, які бачать її вперше:

- **Брендинг** — синя шапка з логотипом «52», title «ЗДО №52 — Адміністрування»
- **Стартова сторінка** — привітання, 4 великих кнопки швидких дій
  («📰 Додати новину», «🖼️ Створити альбом», «💬 Модерувати відгуки»,
  «📁 Додати документ»), жовтий блок з підказками

  зверху welcome + швидкі дії на повну ширину, нижче список секцій
  ліворуч і «Недавні дії» праворуч. На мобільному — стек.
- **Логічно згруповані розділи** з емодзі замість технічних app-name:
  📰 Новини, 🖼️ Фотогалерея, 💬 Відгуки, 🏘️ Групи дитячого садка,
  🎨 Гурткова робота, 👩‍🏫 Спеціалісти, 🏠 Головна сторінка, 📄 Інформаційні
  сторінки, 👨‍👩‍👧 Батьківська сторінка, 🎓 Керівництво закладу,
  📜 Атестація педагогів, 🍽️ Меню харчування, 📁 Документи

Реалізація в [main/admin_helpers.py](main/admin_helpers.py) — функція
`customize_app_list()` перевизначає метод `get_app_list` стандартного
`admin.site` і перегруповує моделі за нашою логічною структурою.

## 🔒 Безпека

- `SECRET_KEY` винесено у `.env` (з fallback для розробки)
- `DEBUG`, `ALLOWED_HOSTS`, `CSRF_TRUSTED_ORIGINS` керуються через `.env`
- Коли `DEBUG=False` автоматично вмикаються:
  - HSTS (`SECURE_HSTS_SECONDS = 31536000`) з `INCLUDE_SUBDOMAINS` і `PRELOAD`
  - `SESSION_COOKIE_SECURE`, `CSRF_COOKIE_SECURE`
  - `X_FRAME_OPTIONS = 'DENY'`
  - `SECURE_BROWSER_XSS_FILTER`, `SECURE_CONTENT_TYPE_NOSNIFF`
  - `SECURE_REFERRER_POLICY = 'same-origin'`
- Захист форми відгуків:
  - Honeypot — приховане поле `website`, яке заповнюють боти
  - Rate-limit — не більше 1 відгуку на 60 секунд з однієї сесії
  - Валідація довжини і діапазону рейтингу на стороні сервера
- `.gitignore` ігнорує `.env`, `db.sqlite3`, `media/`, `media-optimized/`, `__pycache__/`
- `ResilientStaticFilesStorage` — підклас WhiteNoise що не падає з 500-ю
  якщо у маніфесті статики немає файлу (просто віддає не-хешовану URL)

## ♿ Доступність та UX

- **prefers-reduced-motion** — анімації автоматично вимикаються для людей з
  вестибулярними проблемами
- **Lazy-loading** зображень
- **Open Graph мета-теги** — гарні превью при шарингу у Facebook / Telegram / Viber
- **Favicon (inline SVG)** + Apple touch icon
- **Адаптивний слайдер** на головній — на мобільному текст під фото, не поверх
- **Підписи фото** обрізаються до 2 рядків з трикрапкою — не перекривають фото
- **Мобільне меню** з прокручуванням (раніше довгий список «Групи» не вміщався)
- **Telegram/Viber-сумісні** клікабельні `tel:` посилання (автоматично додається префікс +38)

## 🔧 Дрібні (але корисні) технічні рішення

- **Розумні фільтри для контактів** (`main/templatetags/contacts_tags.py`):
  - `split_phones` — розбиває рядок «64-82-55, 64-83-98.» на список телефонів
  - `tel_link` — генерує безпечне `tel:` посилання з префіксом +38
  - `split_hours` — розбиває стіну тексту режиму роботи на буллети
  - `render_map_embed` — автоматично перетворює URL Google Maps на `<iframe>`
  - `address_lines` — розбиває адресу на «індекс + місто» і «вулиця»
- **Дублювання меню на наступний тиждень** через admin action
- **3-формна украінська пагінація** у відгуках з anchor `#reviews` (не стрибає вгору)
- **Кастомні 404 / 500** з кнопкою «На головну»
